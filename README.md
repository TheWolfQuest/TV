# WolfTV 1.0

Personal TV and movie manager for TheWolfQuest.

## Publish on GitHub Pages
Upload every file in this folder to the root of the `TV` repository. GitHub Pages should deploy from the `main` branch and `/ (root)` folder.

Open the first time with: https://thewolfquest.github.io/TV/?v=100

Profiles included: Keith, Debby, Sara, Kevin, John, Glenn.

The shared catalog appears for every profile. Each profile's status, season, episode, rating, favorites, notes, and timeline are stored separately in the browser. Use Backup and Restore to transfer everything between devices.
